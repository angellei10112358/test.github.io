# Radatomu - Text Adventure

## Overview
Single-file HTML text adventure, Dragon Quest–inspired.  
18 locations, 17 enemy types, 4 bosses + final boss, turn-based combat.

## How to Play
Open `index.html` (Chinese) or `index_en.html` (English) in any browser.  
Navigation: click buttons. Combat: Attack / Guard / Item / Flee.

## Game Flow
Town (quest, shops, inn, church) → Forest (slimes, goblins) → 
Bandit Camp → North Forest → Cave B1/B2/B3 → Dragon Peak → 
Wyvern King → Ancient Dragon (final boss)

## Items & Equipment
Weapons: Wood Sword → Copper → Iron → Steel → Flame → Dragon Killer  
Armor: Clothes → Iron → Platinum → Dragon Mail  
Accessories: Power Ring / Guard Ring / Life Ring  
Consumables: Herb, Super Herb, Antidote, Torch, Holy Water, Bomb, Prayer Candle

## Status Effects
- Poison: -3HP/tick, cured by Herb/Antidote/Inn
- Curse: ATK-2 DEF-1, cured at Church (20G)
- Blessing: ATK+2 for 3 battles (Church, 30G)
